# Data Party Logistics

A Machine Learning platform that predicts **vessel ETAs** and **port congestion** using live global shipping data.

## Current capability status

| Capability | Status | Description |
|---|---|
| **Data ingestion** | Built / in progress | AIS, weather, and NOAA ingestion foundations exist in the repo. |
| **Warehouse layer** | Partially built | dbt staging and mart models exist for core analytical datasets. |
| **ML prediction** | Planned | The product direction includes ETA and congestion models, but the repo does not yet show a finished trained model pipeline. |
| **Production API** | Planned | Serving structure exists, but a live product API is not yet implemented end-to-end. |
| **Monitoring** | Early / partial | Reliability and freshness flows exist; model monitoring is still future work. |
| **AI news agent** | Planned | This is still roadmap, not a working surfaced product feature. |
